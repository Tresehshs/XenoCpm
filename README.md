## install python

- ### Termux
    - `pkg update`
    - `pkg install python`
    - `pkg install python-pip`

- ### Linux
    - `sudo apt install python`
    - `sudo apt install python-pip`

- ### Windows
    - Download python3 form [python.org](https://www.python.org/downloads/).
    - if **pip** is not in **PATH** use: `py -3 -m pip install -r requirements.txt`
    - `py -3 main.py`
